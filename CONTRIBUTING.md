# 貢献ガイド

このプロジェクトへの貢献を歓迎します！

## はじめ方

1. このリポジトリをフォークする
2. フィーチャーブランチを作成する (`git checkout -b feature/my-feature`)
3. 変更をコミットする (`git commit -m 'Add my feature'`)
4. ブランチをプッシュする (`git push origin feature/my-feature`)
5. Pull Requestを開く

## コードスタイル

- わかりやすい変数名を使う
- コメントは日本語でも英語でもOK
- 1つのPRには1つの変更をまとめる

## バグ報告

バグを見つけた場合は、Issueを作成して以下の情報を含めてください：
- 再現手順
- 期待される動作
- 実際の動作
- 環境情報（OS、バージョンなど）
